# PWS

## Welkom bij ons PWS!

Het programma kan worden uitgevoerd met
'''
python3 main.py
'''
of
'''
pypy3 main.py
'''
Na het uitvoeren wordt gevraagd een tactiek te kiezen voor zowel speler 1 als voor speler 2. Tot slot wordt het aantal simulaties gevraagd en wordt de code uitgevoerd over alle beschikbare CPU's.
